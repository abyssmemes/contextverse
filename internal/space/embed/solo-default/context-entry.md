# ContextVerse Context Entry

This is a ContextVerse context space. Read files in this order:

1. `identity/me.md` — who you're talking to
2. `team/principles.md` — how we work (respect importance weights)
3. `space-index.md` — what exists in this space
4. `decisions.md` — key decisions and why
5. `projects/<relevant>/project.md` — project context for the current task

Then deep-dive selectively using `team/space-map.md` and the project's `map.md`.

Never modify files outside `identity/` without explicit permission.
